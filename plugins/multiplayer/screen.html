Multiplayer

Full-band multiplayer sessions. Create a room, invite friends, pick instruments, and play together with synchronized audio.

Quick Start

  1. Open the Multiplayer panel from the nav bar
  2. Enter your name and create a room (or join an existing one)
  3. Share the room code with your bandmates
  4. Each player picks an instrument (guitar, bass, drums, or vocals)
  5. Everyone clicks Ready
  6. Host clicks Start -- the song plays with each player's stem muted

How It Works

  • Each player's instrument stem is automatically muted for them
  • Guitar player hears everything except the guitar track
  • Drums player hears everything except the drums track
  • Playback is synchronized via WebSocket time sync
  • Split-screen layout shows each player's highway

Limitations

  • Maximum 4 players per room (one per instrument)
  • All players must have the same song/stems loaded
  • Time sync accuracy depends on network latency
  • For best results, players on the same LAN get tighter sync
